  1. 安装依赖 (已完成)
     pip install -r requirements.txt

  2. 获取 WeChat Key
     企业微信 → 群机器人 → 复制 Webhook URL 中的 key

  4. 查看结果
     企业微信群组会收到推送的新闻消息


【方案 C: 自动化部署】- 30 分钟

  1. 代码推送到 GitHub
     git push

  2. 配置 Secrets
     Settings → Secrets → 添加 WEIXIN_WEBHOOK_KEY

  3. 自动执行
     Actions 每天 09:00 UTC 自动运行
     企业微信自动收到推送
