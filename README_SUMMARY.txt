📌 项目概述
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

AiNews 是一个完整的科技新闻爬虫系统，可以：

  • 🌍 从多个 RSS 源爬取最新科技新闻
  • 📱 自动推送到企业微信群组  
  • ⏰ 通过 GitHub Actions 每天定时自动运行
  • 📊 提供结构化的数据格式
  • 🔄 具有完善的错误处理和重试机制

核心特点: 完全自动化、零维护、开箱即用


🚀 快速开始（3 个步骤）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

方案 A: 快速体验（10 分钟）
  
  1️⃣  运行测试工具
      python test_project.py
      
  2️⃣  运行演示爬虫
      python demo_scraper.py
      
  3️⃣  查看快速参考
      cat QUICK_REFERENCE.md

  ✅ 结果：看到爬虫的实际效果


方案 B: 本地部署（1 小时）

  1️⃣  安装依赖
      pip install -r requirements.txt
      
  2️⃣  获取 WeChat Key
      企业微信 → 群机器人 → 复制 Webhook URL 中的 key
      
  3️⃣  运行爬虫
      set WEIXIN_WEBHOOK_KEY=your_key_here
      python scraper.py
      
  4️⃣  查看企业微信群
      收到推送的新闻消息

  ✅ 结果：完整体验爬虫功能


方案 C: 自动化部署（30 分钟）

  1️⃣  推送到 GitHub
      git push
      
  2️⃣  配置 Secrets
      Settings → Secrets → 添加 WEIXIN_WEBHOOK_KEY
      
  3️⃣  启用 Actions
      Actions 标签 → Enable workflows
      
  4️⃣  每天自动运行
      每天 09:00 UTC 自动执行，企业微信自动收到推送

  ✅ 结果：全自动化，无需人工干预
